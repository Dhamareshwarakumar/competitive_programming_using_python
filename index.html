Competitive Programming Using Python

Description

Competitive programming has always been the most exciting skill amongst the techies. Solving real world problems through software code requires this skill for sure In this regard, for the first time, ACM GMRIT and CSI GMRIT are getting collaborated with each other to present you a workshop on Competitive programming with python from Beginner to Advanced level which also includes an exciting codeathon at the end. You can join beginner stream if you are a beginner or if you are already familiar with basics, you can join advanced stream Come, join us to enhance your programming skills.
